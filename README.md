## Installation

1. Clone the repo
```sh
git clone https://github.com/Eshy10/react-calculator.git
```
2. Open the project Directory in your local machine
```
cd react-calculator.git
```
1. Install NPM packages
```sh
npm install
```
4. Run Application
```JS
npm start
```
